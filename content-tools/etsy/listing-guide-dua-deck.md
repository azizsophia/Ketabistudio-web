Do this once the shop is active. ~10 min. (When the API key approves, I take over
listing automatically — this is just for now.)

## 0. One-time shop setup (if not done)
- Etsy.com -> Sell on Etsy -> open shop, pay the ~$15, set up Etsy Payments
  (bank + ID verification; start early, it lags a day or two).

## 1. Start the listing
Shop Manager -> Listings -> **Add a listing**. Choose **Digital** (not physical)
when asked about the item type.
