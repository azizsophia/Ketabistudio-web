# Etsy listing guide — "A Name Written Into the Qur'an" (personalized name print)

This is the wide-open, first-mover product: a personalized name print whose
hook is the **verified Qur'anic connection** (the cited ayah the name's root
sits in), not just the meaning. Made-to-order: the buyer types the name, we
deliver their custom print. ~10 min to list.

## How "personalized + made-to-order" works on Etsy (important)
Etsy digital listings normally auto-deliver ONE fixed file. For a personalized
name, we can't pre-make every name, so we set it up like this:
1. **Turn ON "Personalization."** Buyer gets a text box to type the name.
2. Upload a small **"How it works" PDF** as the digital file (auto-delivers
   instantly so Etsy is happy). It says: *"Thank you. Your personalized print
   is hand-finished and sent to you here in Etsy Messages within 24 hours."*
3. When the order comes in, we render that name's print and send the real
   high-res files via **Etsy Messages / the order's Files** within 24h.
4. Mark processing time **1 business day**. This keeps reviews clean.

(When the Etsy API key approves, this whole deliver-the-custom-file step gets
automated. Until then it's a 2-minute manual render + send per order.)

## The honesty rule (never break it)
Three tiers, never blurred:
- **"A Name Allah Placed in the Qur'an"** — only for names that literally appear
  (prophets, Maryam, Zayd, surah names).
- **"A Name Written Into the Qur'an"** — for names whose ROOT appears in the
  cited ayah. We say "from the root ___," never "your name is in the Qur'an."
- **"The Meaning of a Name"** — for Arabic names whose root is NOT in the Qur'an
  (e.g. Aws). Meaning + verified heritage note only. NEVER an ayah or a Qur'anic
  claim on these.
This distinction is the trust that lets us charge more than the meaning-only shops.
