# QC CHECKLIST — run on EVERY asset before it is allowed to go live

Standing owner rule (set 2026-07-05, after a reel posted with an end-card
overlap that had to be archived): **nothing posts live until every box below is
checked on the COMPLETE asset, and flagship/one-off pieces are shown to the
owner first, even when she says "post it right away."** The failure was skipping
the final gate under time pressure. The gate is non-negotiable.

## 1. Accuracy (zero false claims)
- [ ] Every factual/religious claim traced to a PRIMARY source. Nothing from memory.
- [ ] Any hadith quoted VERBATIM with its grading, from sunnah.com (or a verified
      mirror like hadithunlocked.com when the proxy blocks sunnah.com).
- [ ] Linguistic/root claims checked against a lexicon or corpus.quran.com; folk
      etymologies rejected (see insight-bank.md DO-NOT-POST list).
- [ ] Sources listed inside the post (essays) or kept on file (reels/carousels).

## 2. Arabic
- [ ] Correct spelling + every diacritic present and correct.
- [ ] Correct right-to-left order and letter shaping.
- [ ] Checked ZOOMED IN on the actual rendered frame, not at thumbnail size.
- [ ] Properly aligned/centered, not clipped.

## 3. Alignment + layout (checked across the ENTIRE timeline)
- [ ] No overlapping text layers at ANY moment. Check every transition.
- [ ] The END of a reel especially: the end card must fully CLEAR the payoff
      line (they must not both be on screen). Watch the final ~3 seconds frame
      by frame.
- [ ] Nothing clipped by the frame edges; consistent centering; safe margins.
- [ ] Legible over the busiest background frame.

## 4. Voice + brand rules (see voice-style.md)
- [ ] NO em dashes anywhere.
- [ ] His/Him/He capitalized for Allah.
- [ ] English-first; Arabic only with transliteration + translation.
- [ ] Faceless imagery only; no identifiable people.
- [ ] Gender framing: insight content universal, mama-framing only on parenting/product.
- [ ] No Star of David / six-pointed star.
- [ ] A clear CTA (save / share / follow / reply-bait / waitlist), rotated.
- [ ] Caption <=2200 chars, <=30 hashtags; reel/Threads text within limits.

## 5. Process
- [ ] Full contact sheet built AND every sampled frame actually scrutinized
      against 1-4 above (do not skim).
- [ ] For reels: sample the hook (0-2s), every card transition, and the final 3s.
- [ ] Shown to owner before going live (flagship/one-off). Scheduled queue items
      were QC'd at build time; re-verify before a slot fires if edited.
